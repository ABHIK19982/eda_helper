# EDA Helper 
## Description 
This agent is designed to help with EDA process on any dataset, where data has been stored as a file. It performs the following tasks 
1. Load a data file. Currently the tool can accept csv,excel and json files only 
2. parses the data and creates plots for each columns in the dataset
3. creates basic pairplots for all columns 
4. Stores all the plots and analysis in a file for furthur analysis. 

This tool has been designed to aid in the task of EDA by performing the basic analysis after which the user can continue.
